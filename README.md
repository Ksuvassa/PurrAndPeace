# PurrAndPeace 🧘‍♀️🐈

A premium, calming Yoga application designed for you and your cat.

## How to Run
Since this application uses modern **Vanilla JavaScript (ES Modules)**, you can run it directly in your browser without any complex build steps.

**Option 1 (Recommended - VS Code Live Server):**
1. Right-click `index.html`.
2. Select "Open with Live Server".

**Option 2 (Direct):**
1. Navigate to the folder `e:\DOCUMENTS\PurrAndPeace`.
2. Double-click `index.html` to open it in your browser.
   *(Note: Some browsers block ES Modules on `file://` protocol. If it doesn't load, use Option 1 or a local server).*

## Features
- **Onboarding**: Personalized profile setup for you and your cat.
- **Harmony Bar**: Watch your connection grow as you practice.
- **Yoga Practice**: 10 daily poses with voice instructions and timer.
- **Tracker & Articles**: Monitor your cat's mood and learn about stress management.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Animations)
- Vanilla JavaScript (ES Modules)
